Reglas del Juego

Objetivo:

Controla la serpiente para comer frutas (puntos amarillos) y crecer, evitando chocar con las paredes o contigo misma.

Controles:

Usa las flechas del teclado (↑ ↓ ← →) o las teclas WASD para mover la serpiente.

Comer Frutas:

Cada fruta comida suma 1 punto y hace crecer la serpiente.

Preguntas:

Cada 3 frutas consumidas, deberás responder una pregunta de opción múltiple para continuar.

Game Over:

El juego termina si la serpiente choca contra las paredes o contra sí misma.

Retroalimentación:

Recibirás información inmediata si tu respuesta es correcta o incorrecta.
